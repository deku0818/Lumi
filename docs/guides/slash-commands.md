# 斜杠命令使用指南

消息以 `/命令名` 开头即触发斜杠命令。desktop 输入框输入 `/` 会弹出补全菜单；飞书里直接发
`/命令`（群里 `@机器人 /命令` 也行）。两端的命令列表都来自后端 `list_commands`，随项目动态
变化，前端不硬编码。

命令后可以接补充说明，如 `/commit 只提交文档改动`。拼错或不存在的 `/xxx` 不报错，按普通
消息交给模型理解。

---

## 内置命令

| 命令 | 可用范围 | 说明 |
|------|------|------|
| `/compact` | 两端 | 强制压缩当前会话历史（保留摘要）。会话太短或末条不是完整回复时提示暂不需要。 |
| `/dream` | 仅 desktop，需启用记忆 | 立即整理记忆：后台综合当前会话 + 其他近期会话。 |
| `/dream-session` | 仅 IM 渠道，需启用记忆 | 整理本会话记忆：只综合当前这个群/私聊的常驻会话。 |
| `/stop` | 仅 IM 渠道 | 停止正在执行的任务（含派生后台任务），清掉排队消息。渠道层即时执行、不排队。 |
| `/clear` | 仅 IM 渠道 | 清空本会话历史，重新开始。 |
| `/help` | 仅 IM 渠道 | 列出全部可用命令（技能命令 / 会话控制两组卡片）。 |

dream 系按载体分流：desktop 短会话只见 `/dream`，IM 长会话（一群/一人一个永久 thread）只见
`/dream-session`——两端各见其一。同名技能会被内置命令屏蔽（内置优先）。

## 技能命令

技能命令从项目 `.lumi/skills/`（及所用 style 的 `skills/`）自动加载。每个技能目录下需包含
`SKILL.md`，定义技能的名称、描述和 prompt。

```
/skill-name [额外文本]
```

额外文本会追加到技能 prompt 后一并发送给 Agent。desktop 与 IM 两端可用同一套技能命令。
