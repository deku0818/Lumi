# 斜杠命令使用指南

Lumi TUI 支持通过 `/` 前缀触发斜杠命令。输入 `/` 后会弹出补全菜单，展示所有匹配的命令。

---

## 内置命令

| 命令 | 说明 |
|------|------|
| `/skills` | 查看所有可用技能列表 |
| `/resume` | 恢复历史会话（需要 checkpoint 持久化） |
| `/cron` | 查看和管理定时任务 |
| `/cron-notify` | 查看定时任务通知记录 |
| `/agents` | 查看所有可用 Agent 列表（Enter 查看详情） |
| `/mcp` | 查看 MCP 服务器状态和工具列表 |
| `/clear` | 清空对话历史，开始新会话 |

---

## 技能命令

技能命令从 `.lumi/skills/` 目录自动加载。每个技能目录下需包含 `SKILL.md`，定义技能的名称、描述和 prompt。

```
/skill-name [额外文本]
```

额外文本会追加到技能 prompt 后一并发送给 Agent。

---

## 会话恢复（/resume）

### 前置条件

需要在 `.lumi/config.json` 中启用持久化存储（`checkpoint` 可选 `sqlite` 或 `postgres`）：

```json
{
  "agents": {
    "checkpoint": "sqlite"
  }
}
```

`memory` 模式下 `/resume` 不可用。

### 使用流程

1. 输入 `/resume` 打开会话选择界面
2. `↑↓` 键选择目标会话，支持搜索过滤
3. `Enter` 确认恢复，`Esc` 取消

---

## 补全菜单

输入 `/` 后自动弹出补全菜单，按前缀模糊匹配。匹配结果超过 12 项时，菜单会自动滑动窗口，始终保持当前选中项可见。

| 按键 | 操作 |
|------|------|
| `↑↓` | 切换选中项 |
| `Tab` / `Enter` | 确认补全 |
| `Esc` | 关闭菜单 |
